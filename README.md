# VMP Style Datapack (1.21.x)

I created a ready-to-upload server folder in:

- `upload-to-server/world/datapacks/vmp_style`

## Install
1. Copy the whole `upload-to-server/world/datapacks/vmp_style` folder into your server world datapacks directory.
2. Start the server or run `/reload`.
3. Use `/trigger ks_menu set 1` for command help.

## Included gameplay style features
- Starter kit (one-time): `/function vmp_style:kit_starter`
- Set & teleport to shared spawn:
  - `/function vmp_style:set_spawn`
  - `/function vmp_style:spawn`
- Set & teleport to shared home warp:
  - `/function vmp_style:set_home`
  - `/function vmp_style:home`
- Random teleport with cooldown:
  - `/function vmp_style:rtp`

> Note: Minecraft version `1.21.11` does not exist as an official release string.
> This datapack targets modern 1.21.x datapack format.
